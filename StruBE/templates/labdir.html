How to acces to the lab directory

First of all, you need a Noor account. If you do not have one yet, please request one here.

When your noor account is created, contact IT Helpdesk to ask for the access to /rcsdata/strube.

Once the access granted, you can read all the files in the directory, add your own, and modify some of them. Depending on your operating system, the access to the lab directory is different.

Noor2
To access the lab directory from Noor2, simply go to /rcsdata/strube. For example :cd /rcsdata/strube
Windows 7
Open your file browser, and click on the top bar.

Then write the address of the directory : \\noor-cifs3.kaust.edu.sa\strube. Press Enter, and, if needed, use your credentials to connect.

OS X
Open your file browser, and click on Go then Connect to Server

Then write the address of the directory : smb://noor-cifs3.kaust.edu.sa/strube. Press Enter, and, if needed, use your credentials to connect.

Linux / others

If you use something else, I guess you know what to do with that : smb://noor-cifs3.kaust.edu.sa/strube
